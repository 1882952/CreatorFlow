# CreatorFlow 启动说明

## 一键启动（推荐）

双击项目根目录下的脚本即可：

- **启动**：双击 `start-all.bat`
  - 自动启动编排服务（端口 18688）+ 前端服务（端口 8080）
  - 首次运行会自动安装 Python 依赖
  - 启动完成后自动打开浏览器

- **停止**：双击 `stop-all.bat`，或在启动窗口按任意键停止

## 手动启动

### 1. 启动编排服务

```bash
cd services/orchestrator
pip install -r requirements.txt   # 首次运行
python main.py
# 服务运行在 http://localhost:18688
```

### 2. 启动前端

```bash
# 方式A：直接打开
creatorflow/index.html

# 方式B：静态服务器
python -m http.server 8080 --directory creatorflow
# 访问 http://localhost:8080
```

## 端口说明

| 服务 | 端口 | 说明 |
|------|------|------|
| 前端 | 8080 | CreatorFlow 页面 |
| 编排服务 | 18688 | 后端任务编排 API |
| ComfyUI | 8188 | 视频生成（需单独启动） |

## 日志

服务日志位于 `logs/` 目录：
- `logs/orchestrator.log` — 编排服务日志
- `logs/frontend.log` — 前端服务日志
